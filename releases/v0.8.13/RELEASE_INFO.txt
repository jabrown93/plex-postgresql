========================================
Plex PostgreSQL Shim - Release v0.8.13
========================================

LOG NOISE REDUCTION: Shim diagnostics now logged at DEBUG

✅ Default logs no longer spammed
✅ Debug tracing still available via PLEX_PG_LOG_LEVEL=DEBUG
✅ No behavioral changes to DB mappings

Quick Install:
1. Extract tarball
2. Stop Plex
3. Copy db_interpose_pg.dylib to installation dir
4. Export PLEX_TOKEN="your_token"
5. Start Plex with DYLD_INSERT_LIBRARIES
6. Run test_aggregate_decltype.sh (optional)

See INSTALL.md for detailed instructions.

Requirements:
- macOS ARM64 (M1/M2/M3)
- Plex Media Server 1.42.x
- PostgreSQL 15.x

Support: https://github.com/cgnl/plex-postgresql
